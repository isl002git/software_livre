GIT na Prática

Instalação

Para começar a trabalhar com git, primeiramente devemos fazer a instalação dele no seu computador. É possível instalar o git tanto no Linux, afinal ele foi desenvolvido pelo próprio Linus Torvalds, assim como no Mac e no Windows.

Para começar, vamos deixar aqui como fazer a instalação dele e, mesmo que você tenha feito isso anteriormente, é importante manter o GIT atualizado.

No Linux

Se você usa uma distribuição baseada em Debian como o Ubuntu, use o apt-get:

sudo apt-get install git

Agora caso você esteja utilizando o Fedora, você pode usar o yum:

sudo yum install git

Agora caso as opções acima não sejam seu caso, utilize este link para encontrar o que precisa:

http://git-scm.com/download/linux

No Mac

Agora no Mac, você tem algumas outras opções.

Caso esteja utilizando o gerenciador de pacotes Homebrew:

brew install git

Utilizando MacPorts:

sudo port install git

Mas, assim como no Linux, caso nenhuma das opções acima seja para você, utilize este link para localizar sua forma de instalação:

https://git-scm.com/download/mac

No Windows

Para fazer a instalação no Windows o processo é bem simples. Basta acessar o site abaixo e baixar a versão referente ao seu SO.

https://git-scm.com/download/win

Mas caso você deseje utilizar o winget para obter o git, abra o terminal como administrador e execute o comando abaixo:

winget install --id Git.Git -e --source winget

Configuração Inicial do Git

Após a instalação do git você deve fazer a configuração inicial básica para utilizar o sistema.

O git contém uma ferramenta chamada git config que permite que você configure algumas variáveis que orientarão o funcionamento dele.

No Linux você pode localizar essa ferramenta em /etc/gitconfig

No macOS, o arquivo .gitconfig geralmente está localizado no diretório home do usuário. ~/gitconfig

No Windows em C:\Users\[usuario]
